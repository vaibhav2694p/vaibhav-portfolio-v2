# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/vaibhav2694p/vaibhav-portfolio-v2.git
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Build production

```bash
npm run build
```

Preview production

```bash
npm run preview
```

Lint

```bash
npm run lint
```
